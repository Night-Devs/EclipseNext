## О проекте
- Eclipse это самодостаточный Discord бот с открытым исходным кодом, простой в использовании и написанный на JavaScript.

### Установка

```bash
$ npm install
```

### Запуск приложения

```bash
# Разработка
$ npm run dev

# Продакшн
$ npm run start
```
